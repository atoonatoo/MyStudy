##### 제목 : 

   - 메인 메서드
        
        ```java
        - public static void main(String[] args) : 
        	"main 함수는 자바의 시작점이다."
        	"JAVA 소스는 무조건 main 함수에서부터 시작한다."
        	"이것은 JAVA의 규칙이다."
        	자바가상머신(JVM)은 main이라는 이름을 가진 메서드를 찾아 프로그램을 시작한다.
        	
        	* public : JVM이 main함수를 찾을수 있도록 해준다.
        		public은 접근제어자이다. 
        		메인 메서드는 자바의 시작점이기에 어디에 있건 JVM이 접근하는데 
        		제한이 없어야하기에 public을 접근제어자로 선언하는 것이다.
        	
        		% 접근제어자 : public > protected > default > private 
        			왼쪽에서 오른쪽으로 갈수록 변수나 메서드 등에 대한 접근이 어렵게 된다.
        	
        			1) public : 접근에 제한이 없음.
        			2) protected : 동일패키지 내에 있거나, 파생패키지의 경우 접근 가능.
        			3) default : 아무런 접근제어자를 명시 하지않았을 경우 할당되는 제어자이며, 
        				 동일한 패키지 내에서만 접근 가능.
        			4) private : 자기 자신의 클래스 내에만 접근 가능.
        
        	* static : JVM이 main함수를 곧 바로 시작할 수 있도록 해준다.
        		static은 "정적인, 움직이지 않는"이라는 의미이다.
        		
        		우선 한가지 짚고 넘어가자면 Class라는 것은 객체를 만들기 위해 필요한 것이다.
        		하지만 Main Class는 그 안에 메인 메서드만 존재할 뿐, 객체를 생성하지 않는다.
        		그 이유는 메인 메서드에 붙어있는 "static이 작용"하기 때문이다.
        		static은 "메모리에 고정된다, 변하지않는다"라는 의미를 가지고 있다.
        		하지만 실제로는 이 static은 '공유'의 의미를 가지고 있다.
        		static이 붙어있으면 모든 객체가 공유하는 변수, 메서드를 의미한다.
        		
        		여기서 잠깐,
        		생성되는 모든 객체가 공유하는(모든 객체에서 공유하는 동일하게 작동하는) 
        		메서드를 사용하기 위해서, 객체를 하나 생성할 필요가 있는것인가? 
        		"없다."
        				
        
        	* void : main함수가 종료되면 프로그램도 종료되므로 return값이 필요하지 않다.
        	* String[] args : 커맨드라인 등을 통해. main함수 내부에서 사용할 수 있는 
        		String 데이터를 전달할 수 있다.
        	
        - 상세 : 
        
        1. 모든 Java 프로그램은 main 함수(메서드)에서 시작한다.
        
        2. 메인 메서드는 반드시 public, static, void로 선언하고, 
        	 그 매개변수는 String 배열 타입으로 선언해야 한다.
