##### React Next.js 14와 Spring Boot를 활용한 라우팅 및 데이터베이스 연동

    - **TS**: 통신과 관련된 로직만 담당.
- **DB 등록**: 화면에서 입력된 데이터를 통해 API를 호출하여 Ncloud의 DB에 데이터를 등록.
