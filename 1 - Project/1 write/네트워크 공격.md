# 1. 공격받는 네트워크

공격받는 네트워크란 악의적인 사용자가 네트워크의 보안 취약점을 이용하여 침입하거나 악성 코드, 피싱, Ddos(분산 서비스 거부 공격) 등의 방법으로 시스템을 손상시키고, 정보를 탈취하거나 네트워크를 마비시키는 상태를 의미한다.
