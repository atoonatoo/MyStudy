##### 제목 : 

    - 웹 서버
        
        ```java
        - 두대의 컴퓨터를 물리적으로 연결하려면? : 
        	일반적으로 이더넷 케이플을 사용하며 무선으로는 WIFI나 블루투스를 사용하여 
        	연결한다.
        	
        	이러한 네트워크 연결은 두 대의 컴퓨터로 제한되지않는다. 
        	원하는 만큼의 컴퓨터를 연결할 수 있다. 하지만 금방 복잡해질 것이다.
        	예를 들어 컴퓨터 10대를 연결하려면 컴퓨터당 9개의 플래그가 포함된 
        	45대의 케이블이 필요하다.(...)
        
        - 여러대의 컴퓨터를 연결하려면? : 
        	이 문제를 해결하기 위해서 네트워크의 각 컴퓨터는 라우터라고 하는 특수한
        	작은 컴퓨터에 연결된다.	
        	라우터에는 단 하나만의 작업이 있다.
        	기차역의 신호원처럼 특정 컴퓨터에서 보낸 메세지가 올바른 대상의 컴퓨터에
        	잘 도착하는지 확인하는 역할이다.
        	컴퓨터B로 메세지를 보내려면 컴퓨터A가 라우터로 메세지를 보내야하며, 
        	라우터는 해당 메세지를 컴퓨터B로 전달하고 메세지가 컴퓨터C로 전달되지 않도록
        	해여 한다.
        	시스템에 라우터를 추가하면 10대의 컴퓨터로 구성된 네트워크에는
        	10개의 케이블만 필요하다. 
        	각 컴퓨터에 대한 단일 플러그와 10개의 플러그가 있는 라우터이다.
        
        - 수백..수천대의 컴퓨터를 연결하려면..? : 
        	여태까지는 그런데로 잘 되었다.. 하지만 수백, 수천, 수만의 컴퓨터를 연결하려면
        	어떻게 해야할 것인가?
        	물론 단일 라우터로는 확장할 수 없다. 
        	
        	*	하지만 자세히 읽어보면 
        	  다른 라우터도 다른 컴퓨터와 마찬가지로 컴퓨터라고 했는데 두 라우터를 함께
        		연결하지 못하게 하는 이유는 무엇인가?
        	* 컴퓨터를 라우터에 연결한 다음 라우터를 라워에 연결하면 무한히 확장할 수 있다.
        ```
